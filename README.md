Hotel Management System (C Programming)
A robust, console-based application built in C to manage daily hotel operations, inventory, and guest transactions.

🚀 Features
Administrator Module
Room Configuration: Define room types (King, Queen, Double, Single), set availability, and establish daily rates.

Inventory Management: Add, update, or delete Food & Beverage items and hotel facilities.

Reception Module
Dynamic Check-In: Book rooms, specify stay duration, and add optional services (food/facilities) to the guest's account.

Automated Billing: Generates formatted receipts with sub-totals and a grand total.

Payment Processing: Supports Cash and Credit Card payment simulations with change calculation.

Check-Out: Frees up rooms automatically, making them available for the next guest.

🛠️ Technical Highlights
Dynamic Tracking: Real-time updates of room stocks and financial totals.

Input Validation: Built-in loops to prevent invalid entries (negative prices, overbooking, or empty names).

Clean Code: Structured using modular functions for easy maintenance.

💻 How to Run
Compile: Use any C compiler (e.g., GCC).

Execute:

Setup: Navigate to the Administrator Module first to initialize room stocks before performing a Check-In.
